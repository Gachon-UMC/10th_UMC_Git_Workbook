## git 기본 배우기 
- git add 파일이름 : 파일 스테이징
- git commit -m "메세지" : 커밋
- git push: 원격 레포로 푸쉬
- git clone:원격 레포 연결
- git status: 현 작업장소 상태 알려주기
- git log: 커밋 기록 확인

***.ignore이란?*** 파일은 워킹 디렉토리 내에서 추적되지 않아야 하는 파일 목록을 지정하는 파일
- touch  .gitignore: .ignore파일 생성, 이 파일에 적힌 파일, 폴더 등은 추적되지않음

***branch이란?*** 내가 작업하는 독립적인 영역
- git branch "브랜치 이름": 브랜치 생성
- git switch "브랜치 이름":브랜치 이

***내 브랜치를 메인이 반할 때는?***
- pull request(pr): git hub레포지토리 주소에서 pull requst-> new pull request
